# Clasper Trademark Guidelines

This repository is Apache 2.0–licensed, but the **Clasper** and **Clasper Cloud** names and logos are trademarks.
These guidelines prevent confusion and bad-faith SaaS clones.

## Allowed (without permission)

- Referencing **Clasper** to describe this project
- Using **Clasper** in accurate, non-confusing statements (e.g., “works with Clasper Core”)
- Forking the code **if you rename** the project and remove Clasper branding

## Not Allowed (without explicit permission)

- Using **Clasper** or **Clasper Cloud** as the name of a product or service
- Branding a hosted service as “Clasper Cloud” or similar
- Using Clasper logos, wordmarks, or confusingly similar branding

## Forks Must Rename

If you fork this repository and redistribute it, you **must**:

- choose a new name that does not include “Clasper”
- remove or replace Clasper branding and logos
- avoid implying affiliation with Clasper Cloud

Questions or permission requests: contact the project owner.
